QA Item 1: Players start glitching after game launches

Details:

  • Reported By: Omar Al-Guthmy
  • Date: 31st January, 2023
  • Game Version: 1.0.2
  • Platform: Windows 10, 64 bit
  • Priority: High
  • Severity: Major
  • Status: New/Open

Description:

When players are loaded into the game after launch, they start glitching and teleporting. The players are supposed to have fluid grounded movements.

Steps to Reproduce:

  1. Launch the game.
  2. Join a multiplayer match.
  3. Observe other players' movements.

Expected Result:

Players should have fluid, grounded movements without any glitches or teleportation.

Actual Result:

Players are glitching and teleporting after they are loaded into the game.

QA Item 2: Items appear unpicked to other players after one player picks them up

Details:

  • Reported By: Omar Al-Guthmy
  • Date: 15th February, 2023
  • Game Version: 1.0.2
  • Platform: Windows 10, 64 bit
  • Priority: High
  • Severity: Major
  • Status: New/Open

Description:

When a player picks up an item, it disappears for them as expected. However, the item remains visible to other players, appearing as if it has not been picked up. This issue occurs consistently across all item types and pickup events.

Steps to Reproduce:

  1. Start the game and join a multiplayer match.
  2. Locate an item on the ground.
  3. Pick up the item.
  4. Ask another player to confirm whether they still see the item on the ground.

Expected Result:

Once an item has been picked up by a player, it should disappear for all players in the game.

Actual Result:

After an item has been picked up by a player, it remains visible to other players in the game.

QA Item 3: Character animations not visible to other players

Details:

  • Reported By: Omar Al-Guthmy
  • Date: 5th March, 2023
  • Game Version: 1.0.2
  • Platform: Windows 10, 64 bit
  • Priority: High
  • Severity: Major
  • Status: New/Open

Description:

When a player performs an action that triggers an animation, the animation is only visible to them. Other players see the character as floating and moving without the associated animation. This bug affects all animations and actions across the game.

Steps to Reproduce:

  1. Start the game and join a multiplayer match.
  2. Perform an action that should trigger an animation.
  3. Ask another player to confirm whether they can see the animation.

Expected Result:

All player animations should be visible to all players in the game.

Actual Result:

Player animations are only visible to the player performing them, other players see the character as floating without animation.

QA Item 4: Shared Inventory UI Across Players

Details:

  • Reported By: Omar Al-Guthmy
  • Date: 27th February, 2023
  • Game Version: 1.0.2
  • Platform: Windows 10, 64 bit
  • Priority: High
  • Severity: Major
  • Status: New/Open

Description:

The game's inventory system appears to be shared across players in a multiplayer game. When one player adds or removes an item in their inventory, the same change is reflected in the inventories of all other players.

Steps to Reproduce:

  1. Start the game and join a multiplayer match.
  2. Pick up an item to add it to the inventory.
  3. Ask another player to open their inventory and confirm whether the item appears in their inventory as well.

Expected Result:

Each player should have an individual inventory. Changes made by one player to their inventory should not affect the inventories of other players.

Actual Result:

Changes to one player's inventory are reflected across the inventories of all other players in the game.

QA Item 5: Incorrect Dumpster Exit Teleportation

Details:

  • Reported By: Omar Al-Guthmy
  • Date: 6th April, 2023
  • Game Version: 1.0.2
  • Platform: Windows 10, 64 bit
  • Priority: High
  • Severity: Major
  • Status: New/Open

Description:

When a player hides in a dumpster by pressing a specific key, they are correctly teleported inside the dumpster. However, when they exit, they are incorrectly teleported to a different dumpster exit instead of the one they entered.

Steps to Reproduce:

  1. Start the game and join a multiplayer match.
  2. Find a dumpster and press the specified key to hide inside.
  3. Press the key to exit the dumpster.
  4. Observe the location of the exit.

Expected Result:

Players should exit the dumpster at the same location they entered.

Actual Result:

Players are teleported to a different dumpster exit when they exit the dumpster.

QA Item 6: Collective UI Coin Score Update

Details:

  • Reported By: Omar Al-Guthmy
  • Date: 20th April, 2023
  • Game Version: 1.0.2
  • Platform: Windows 10, 64 bit
  • Priority: High
  • Severity: Major
  • Status: New/Open

Description:

The game's coin score UI updates collectively for all players instead of individually. When a player earns or loses coins, the change is reflected in the coin scores of all players, not just the one who actually earned or lost the coins.

Steps to Reproduce:

  1. Start the game and join a multiplayer match.
  2. Earn or lose coins.
  3. Ask other players to confirm whether their coin score UI has also updated.

Expected Result:

The coin score UI should update individually for each player, reflecting only the coins they have earned or lost.

Actual Result:

The coin score UI updates collectively for all players, regardless of who actually earned or lost the coins.

QA Item 7: Bubble Shield Not Updating with Player Movement

Details:

  • Reported By: Omar Al-Guthmy
  • Date: 31st January, 2023
  • Game Version: 1.0.2
  • Platform: Windows 10, 64 bit
  • Priority: High
  • Severity: Major
  • Status: New/Open

Description:

When a player activates the Bubble Shield item, it doesn't move with the player. Instead, it remains stationary at the point of activation, thereby failing to provide the intended protection for the player.

Steps to Reproduce:

  1. Start the game and join a multiplayer match.
  2. Activate the Bubble Shield item.
  3. Move your character.
  4. Observe whether the Bubble Shield moves with the character.

Expected Result:

The Bubble Shield should move with the player when they move, providing continuous protection.

Actual Result:

The Bubble Shield remains stationary at the point of activation and does not move with the player.
