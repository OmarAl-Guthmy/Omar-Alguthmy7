QA Item 2: Items appear unpicked to other players after one player picks them up

Visual Demonstration:

Details:

  • Reported By: Omar Al-Guthmy
  • Date: 15th February, 2023
  • Game Version: 1.0.2
  • Platform: Windows 10, 64 bit
  • Priority: High
  • Severity: Major
  • Status: New/Open

Description:

When a player picks up an item, it disappears for them as expected. However, the item remains visible to other players, appearing as if it has not been picked up. This issue occurs consistently across all item types and pickup events.

Steps to Reproduce:

  1. Start the game and join a multiplayer match.
  2. Locate an item on the ground.
  3. Pick up the item.
  4. Ask another player to confirm whether they still see the item on the ground.

Expected Result:

Once an item has been picked up by a player, it should disappear for all players in the game.

Actual Result:

After an item has been picked up by a player, it remains visible to other players in the game.

  • Date: 6th April, 2023
  • Game Version: 1.0.2
  • Platform: Windows 10, 64 bit
  • Priority: High
  • Severity: Major
  • Status: New/Open

Description:

When a player hides in a dumpster by pressing a specific key, they are correctly teleported inside the dumpster. However, when they exit, they are incorrectly teleported to a different dumpster exit instead of the one they entered.

Steps to Reproduce:

  1. Start the game and join a multiplayer match.
  2. Find a dumpster and press the specified key to hide inside.

Players are teleported to a different dumpster exit when they exit the dumpster.
